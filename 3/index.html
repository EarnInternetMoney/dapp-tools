Tools for dapps

Developing for Ethereum demands a suite of tools to help with construction, correctness, deployment, and management.

The open source toolkit by DappHub consists of

  • dapp, a command line tool that initializes, compiles, links, tests, deploys, and interacts with Ethereum contracts;
  • seth, a utility knife for slicing and dicing transactions, performing remote calls, and converting between data formats; and
  • hevm, an EVM implementation designed for testing and verification, with a nimble debugger that runs in the terminal.

We prefer tools that are fast, transparent, and good citizens of the command line, and that's how we've built the dapp suite.

These tools grew out of our work on Maker and other consulting projects.

Installing the DappHub toolkit

We recommend using the Nix package manager to install these tools. It works on Linux and OS X, and we maintain a channel with all our software along with the Solidity compiler (several versions). 

# Install Nix:
curl https://nixos.org/nix/install | sh

# Add the DappHub channel to Nix:
nix-channel --add https://nix.dapphub.com/pkgs/dapphub
nix-channel --update

# Install the various developer tools:
nix-env -iA dapphub.{dapp,hevm,seth,solc}

Learning

These tools have been satisfying our own needs, and now we're getting them ready for everyone to use.

Until we publish learning material—manuals, videos, tutorials—the best way to learn is to install the tools and then look at a project like ds-token.

You should also join our chat where we'll be able to help you get started.
