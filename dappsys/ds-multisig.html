The ds-multisig library is DSGroup with a command-line interface. A list of members, the required quorum and the window of time in which actions must be approved are fixed when the DSGroup contract is created. Actions can then be proposed, confirmed and triggered once a group quorum has been reached.

Installation & deployment

Install dapp to build and deploy the contract:

{% highlight bash %} dapp build dapp deploy DSGroup '[ 0011111111111111111111111111111111111111, \ 0022222222222222222222222222222222222222, \ 0033333333333333333333333333333333333333 \ ]' 2 86400 {% endhighlight %}

Install the seth dependency in order to use the command line interface. Then type make from the ds-multisig directory to install the ds-group CLI tool:

{% highlight bash %} Usage: ds-group [] or: ds-group --help Commands: action print information about a multisig action confirm confirm a proposed multisig action ls list already-proposed multisig actions propose propose a new multisig action trigger trigger a confirmed multisig action verify verify the meaning of a multisig action {% endhighlight %}
