We recommend using the Nix package manager. It works on Linux and OS X and is completely unobtrusive. We maintain our own channel for distributions of our software along with the Solidity compiler (multiple versions). 
  # Install Nix:
  curl https://nixos.org/nix/install | sh

  # Add the DappHub channel to Nix:
  nix-channel --add https://nix.dapphub.com/pkgs/dapphub
  nix-channel --update

  # Install one or many tools as required:
  nix-env -iA dapphub.{dapp,hevm,seth,solc}
