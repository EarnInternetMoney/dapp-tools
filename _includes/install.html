We recommend using the Nix package manager. It works on Linux and OS X and is self-contained and unobtrusive. 

  # Install Nix:
  curl https://nixos.org/nix/install | sh

  # Add the DappHub channel to Nix:
  nix-channel --add https://nix.dapphub.com/pkgs/dapphub
  nix-channel --update

  # Install one or more tools:
  nix-env -iA dapphub.{dapp,hevm,seth,solc}
