DappHub

DappHub is a community of developers working on blockchain software. We maintain a suite of Unix tools designed for people who are building and maintaining distributed applications, mostly related to Ethereum.

To install the tools on your computer, run the following two commands: 

# Install the Nix package manager:
curl https://nixos.org/nix/install | sh
# Install the Dapp and Seth programs:
nix-env -i dapp seth

Dapp

Our main developer tool is called Dapp. It lets you do things like:
  • Initialize a new Ethereum application project
  • Build an Ethereum application (i.e., invoking the compiler)
  • Run unit tests written in Solidity
  • Deploy an application to the Ethereum blockchain
  • Send transactions to a deployed application

Seth

Another tool is called Seth. It enables use cases like the following:
  • Send an arbitrary Ethereum transaction
  • Query the Ethereum blockchain for a piece of data
  • Interact with your local Ethereum client
  • Convert data between different formats
  • Sign data using your Ethereum private key
