Basics

Quick intro to working with the dapp tools

You can get started with dapp development very easily: 

$ mkdir example
$ cd example
$ dapp init
$ vi src/example.sol
$ dapp test
$ dapp debug

For details about the dapp command, see dapp.tools/dapp.
