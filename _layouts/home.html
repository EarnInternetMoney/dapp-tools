Introduction to the Nix package manager

The dapp tool builds on Nix, a package manager with properties that make it uniquely robust, far beyond tools like npm, and uniquely self-contained.

You don't need knowledge of Nix to operate dapp, but we recommend anyone interested in system administration or software deployment to learn about it.

Nix's robustness is due to its use of cryptographic digests to track dependencies and configurations. This fits naturally with the blockchain paradigm â€” we're in the age of hashes.
